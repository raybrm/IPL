Les nuages

cirrus

Ce type de nuage possède les espèces suivantes:

  • flocus
  • castellanus*
  • spissatus
  • uncinus
  • fibratus

Pas d'hydrométéores précisés.

Ce nuage (cirrus) culmine minimum 6000 mètres et maximum à 12000 mètres.

cirrocumulus

Ce type de nuage possède les espèces suivantes:

  • lenticularis
  • stratiformis
  • flocus
  • castellanus*

Pas d'hydrométéores précisés.

Ce nuage (cirrocumulus) culmine minimum 5000 mètres et maximum à 7000 mètres.

altocumulus

Ce type de nuage possède les espèces suivantes:

  • lenticularis
  • stratiformis
  • castellanus*
  • flocus

Pas d'hydrométéores précisés.

Ce nuage (altocumulus) culmine minimum 2000 mètres et maximum à 6000 mètres.

cumulus

Ce type de nuage possède les espèces suivantes:

  • fractus
  • humilis
  • mediocris
  • congestus

Pas d'hydrométéores précisés.

Ce nuage (cumulus) culmine minimum 200 mètres et maximum à 8000 mètres.

strato-cumulus

Ce type de nuage possède les espèces suivantes:

  • stratiformis
  • lenticularis
  • castellanus*

Pas d'hydrométéores précisés.

Ce nuage (strato-cumulus) culmine minimum 0 mètres et maximum à 2000 mètres.

cumulonimbus

Ce type de nuage possède les espèces suivantes:

  • calvus
  • capillatus
  • mammatus

Hydrométéores : Averses (parfois de neige, de grésil ou de grêle).

Ce nuage (cumulonimbus) culmine minimum 300 mètres et maximum à 18000 mètres.

* L'espèces castellanus est une espèce de 4 nuages différents.
